# 🍎 Farmers Fruit Market

A full-stack e-commerce web application built with Django that allows users to browse fresh produce through a clean, responsive interface. The application includes an administrative dashboard for managing products and is deployed online using Render.

## 🌐 Live Demo

https://django-farmers-fruit-market-2.onrender.com/products/#

## ✨ Features

- Browse products with images, prices, and stock availability
- Django Admin dashboard for product management
- Automatic database seeding using a custom management command
- Responsive Bootstrap user interface
- Production deployment with Render
- GitHub integration with automatic deployments

## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/Alishba-hyphen/django-Farmers-Fruit-Market.git
cd django-Farmers-Fruit-Market
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run migrations:

```bash
python manage.py migrate
```

Seed the database:

```bash
python manage.py seed_products
```

Start the server:

```bash
python manage.py runserver
```

Visit:

https://django-farmers-fruit-market-2.onrender.com/products/#
